# Mean-Variance-Standard Deviation Calculator

This project is part of the freeCodeCamp **Data Analysis with Python** certification.

## Description
A Python function that takes a list of 9 numbers, converts it into a 3x3 NumPy array, and returns the mean, variance, standard deviation, max, min, and sum across rows, columns, and the flattened matrix.

## Technologies
- Python
- NumPy
